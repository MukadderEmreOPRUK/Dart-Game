Başlangıç
---------

Projeyi çalıştırmak için aşağıdaki adımları takip edebilirsiniz.

1.  Proje dosyalarını indirin veya klonlayın.

bashCopy code

`git clone https://github.com/KULLANICI_ADI/PROJE_ADI.git`

1.  Dosyaları bir web tarayıcısında açın.

arduinoCopy code

`cd PROJE_ADI
open index.html`

1.  Dart tahtası sayfası açılacaktır ve oyuna başlamak için "Add Player" butonuna tıklayarak oyuncuları ekleyebilirsiniz. Tıklanan alanın puanı, oyuncunun puanına eklenir ve sıradaki oyuncu otomatik olarak seçilir.

Oyuncu Ekleme ve Çıkarma
------------------------

Oyuncuları eklemek ve çıkarmak için aşağıdaki adımları izleyebilirsiniz.

### Oyuncu Ekleme

1.  "Add Player" butonuna tıklayın.
2.  Adını yazın ve "Add" butonuna tıklayın.

### Oyuncu Çıkarma

1.  "Remove Player" butonuna tıklayın.
2.  Oyuncunun adını seçin ve "Remove" butonuna tıklayın.

Puan Hesaplama
--------------

1.  Oyuncunun adını seçin.
2.  Dart tahtasında istediğiniz bölgeye tıklayın.
3.  Tıklanan bölgenin puanı, oyuncunun puanına otomatik olarak eklenir ve sıradaki oyuncu seçilir.

Desteklenen Bölgeler
--------------------

Aşağıdaki bölgeler tıklanabilir ve puanları oyuncunun puanına otomatik olarak eklenir.

-   İç çember: 2x puan
-   Dış çember: 1x puan
-   İç üçgen: 3x puan
-   Dış üçgen: 2x puan
